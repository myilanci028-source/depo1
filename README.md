# Yılancıoğlu Barkod Klavye

Android için kamera destekli barkod klavye test projesi.

- Çevrimdışı bundled ML Kit modeli
- CameraX önizleme, dokunarak odaklama, yakınlaştırma ve fener
- EAN-13/EAN-8, UPC-A/UPC-E, Code 128/39/93, Codabar, ITF, QR, Data Matrix, PDF417 ve Aztec
- Okunan değeri aktif metin alanına doğrudan yazma
- İnternet izni yoktur

APK, GitHub Actions tarafından test amacıyla otomatik oluşturulur.
