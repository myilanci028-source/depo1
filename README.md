# Yılancıoğlu Barkod Klavye V2

V1 sürümüne dokunmadan, ayrı kurulabilen gelişmiş Android barkod klavyesi.

## V2 yenilikleri

- Ayrı uygulama kimliği: V1 ve V2 aynı telefonda birlikte çalışır
- Enter, Tab, Boşluk veya Hiçbiri sonlandırma seçeneği
- Ses ve titreşim ayrı ayrı açılıp kapatılabilir
- Seri okuma modu
- Aynı barkodu art arda yazmaya karşı koruma
- 1×, 2× ve 3× hızlı yakınlaştırma
- Son 20 okumanın cihaz içi geçmişi
- Yeni lacivert-altın arayüz ve ayrı V2 simgesi
- İnternet izni yoktur; tarama ve geçmiş cihazda kalır

## Desteklenen barkodlar

EAN-13, EAN-8, UPC-A, UPC-E, Code 128, Code 39, Code 93, Codabar, ITF, QR, Data Matrix, PDF417 ve Aztec.

APK, `barcode-keyboard-v2` dalında GitHub Actions tarafından otomatik oluşturulur.
